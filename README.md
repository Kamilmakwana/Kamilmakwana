# 🚀 Featured Project

## 🔄 GitForkSync

**GitForkSync** is a modern developer tool that helps synchronize GitHub forked repositories with their upstream repositories in one place.

Built using **Next.js**, it provides a clean, responsive interface for developers who maintain multiple forks.

### 🌐 Live Demo

👉 **https://www.gitforksync.com/**

### 💻 Source Code

👉 **https://github.com/Kamilmakwana/GitForkSync**

### Features

- 🚀 Sync GitHub forks

- ⚡ Fast & Modern UI
